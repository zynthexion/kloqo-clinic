# **App Name**: MediDash

## Core Features:

- Dashboard Overview: Display key metrics such as total invoices, patient count, appointment numbers, and bedroom occupancy.
- Patient Overview Chart: Display interactive charts presenting patient distribution by age and department using mock data.
- Doctor Availability: Provide a tool that displays each doctor's schedule, with AI reasoning used to dynamically adjust displayed availability.
- Appointment Scheduling: Offer the ability to schedule patient appointments. For the MVP, appointment booking is recorded locally without actual scheduling.
- Real-time Updates: Simulate real-time updates using JavaScript, reflecting immediate changes in reports (room cleaning, equipment maintenance, etc.).
- Calendar Visualization: Visually display appointments on a calendar, with the current day highlighted, using only mock data.
- Recent Activity Feed: Display recent activities such as patient discharges, admissions, and equipment maintenance updates using local storage to mimic activity updates.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) for a calm and professional feel.
- Background color: Light gray (#F5F5F5) for a clean, modern aesthetic.
- Accent color: Teal (#26A69A) to highlight key interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif, blends modernity with a touch of personality.
- Use consistent and modern icons to represent different sections and actions. Use the heroicons library.
- Use a clean, grid-based layout to present information clearly and logically.
- Incorporate subtle animations, like smooth transitions when updating the dashboard.
